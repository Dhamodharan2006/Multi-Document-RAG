# 📚 Multi-Document RAG System

Welcome to the **Multi-Document RAG System** for Academic Research Papers!

## What can I do?

- **Upload** PDF, DOCX, or TXT documents for analysis
- **Query** your documents with natural language questions
- **Compare** findings across multiple research papers
- **Evaluate** the system using RAGAS metrics

## Commands

| Command | Description |
|---------|-------------|
| `/docs` | List all ingested documents |
| `/compare` | Compare two documents |
| `/eval` | Run RAGAS evaluation |
| `/help` | Show available commands |

To ingest a new document, simply attach a PDF, DOCX, or TXT file to your message!

Simply type a question to search across your ingested documents!
